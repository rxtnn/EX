# Водить.РФ — портал записи на курсы вождения речного транспорта

Демонстрационный экзамен 2026, вариант В1 (09.02.07-3-2026-ПУ).

## Стек

- PHP 8 (PDO MySQL)
- MySQL / phpMyAdmin
- Bootstrap 5, Bootstrap Icons
- JavaScript без фреймворков
- Шрифт Rubik (Google Fonts)

## Установка

1. Распаковать проект в `htdocs/exam/` (XAMPP / OSPanel / Open Server).
2. Импортировать `install.sql` через phpMyAdmin — будет создана БД `vodit_rf`.
3. При необходимости отредактировать `includes/db.php` (логин/пароль MySQL).
4. Открыть в браузере: `http://localhost/exam/`.

## Доступ администратора

- URL: `http://localhost/exam/admin/login.php`
- Логин: `Admin26`
- Пароль: `Demo20`

## Структура

```
exam/
├── install.sql              — схема БД
├── includes/                — db, header, footer, helpers
├── assets/                  — css, js, images
├── admin/                   — панель администратора
├── index.php                — главная со слайдером
├── register.php / login.php — регистрация / вход
├── profile.php              — личный кабинет
├── apply.php                — оформление заявки
└── review.php               — отзыв об обучении
```

## Реализованный функционал

### Основной (модуль 1)
- Регистрация с валидацией (логин 6+ латиница/цифры, пароль 8+, ФИО, дата, телефон, e-mail)
- Авторизация по логину/паролю с информативными ошибками
- Личный кабинет с историей заявок и отзывами
- Подача заявки (тип транспорта, дата, способ оплаты)
- Админ-панель (Admin26 / Demo20) со сменой статуса заявок

### Расширенный (модуль 2)
- Подсказки по полям формы регистрации, мгновенная валидация
- Уведомления при неверных данных в авторизации
- Слайдер на главной и в личном кабинете (4 слайда, авто 3 сек, prev/next, dots)
- Возможность оставить отзыв только после смены статуса
- Выпадающие списки и поле даты в форме заявки

### Оптимизация (модуль 3)
- Адаптивный дизайн (390×844 и шире)
- Микроанимации (fade-in-up, pulse, hover-эффекты)
- Фильтры, сортировка, пагинация и тосты-уведомления в админке
- Использование Bootstrap для технологичности и единообразия
